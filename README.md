# Sistemapostural

Sistema de Correção Postural

Este projeto tem como objetivo monitorar a postura do usuário utilizando sensores acoplados a uma cadeira. Quando uma postura incorreta é detectada, o sistema emite alertas sonoros e visuais.

- Componentes utilizados
Arduino Uno
Botão (simulando sensor)
LED
Buzzer
Resistores


- Funcionamento

O sistema realiza a leitura de um sensor (simulado por botão). Quando acionado, interpreta como postura incorreta e ativa:
LED
Buzzer


- Código
linguagem C/C++.


- Protótipo
O circuito pode ser simulado no Tinkercad.
